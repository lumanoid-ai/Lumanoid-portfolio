# Luminoid AI

Luminoid AI is a premium AI agency website built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion for stunning animations.

## Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- npm (comes with Node.js) or another package manager like yarn/pnpm

## Getting Started

1. **Install dependencies**
   Open your terminal in the project root directory and run:
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```
   This will start the local server. Open the URL provided in your terminal (usually `http://localhost:5173`) in your browser to view the app.

## Available Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production to the `dist` folder.
- `npm run preview`: Locally previews the production build.
- `npm run lint`: Runs ESLint to catch errors and enforce code style.

## Technologies Used

- **Framework:** React 19 + TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion, GSAP, Lenis (smooth scrolling)
- **3D Graphics:** Three.js, React Three Fiber
- **Icons:** Lucide React
- **Routing:** React Router DOM
